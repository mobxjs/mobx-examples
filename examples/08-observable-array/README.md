# 08 — Observable Array

Demonstrates MobX observable arrays and their proxy behavior.

## Concepts

- **Observable arrays** — arrays wrapped in a Proxy that tracks all mutations
- **Standard methods** — `push`, `splice`, `map`, `filter`, `find` all work as expected
- **`toJS()`** — converts an observable array back to a plain JavaScript array

## Run locally

```bash
npm install
npm run dev
```
