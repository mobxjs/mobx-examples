# 06 — When

Demonstrates `when` — a one-time reactive trigger.

## Concepts

- **Callback form** — `when(predicate, effect)` runs the effect once when the predicate becomes true
- **Promise form** — `await when(predicate)` for use with async/await
- **Auto-disposal** — `when` automatically cleans up after firing

## Run locally

```bash
npm install
npm run dev
```
