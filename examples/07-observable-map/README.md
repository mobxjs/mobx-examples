# 07 — Observable Map

Demonstrates `observable.map` for reactive key-value collections with dynamic keys.

## Concepts

- **`observable.map`** — creates a reactive Map that tracks additions, updates, and deletions
- **Dynamic keys** — unlike plain objects, maps can reactively track new keys being added
- **Map API** — `set()`, `get()`, `has()`, `delete()`, `size`

## Run locally

```bash
npm install
npm run dev
```
